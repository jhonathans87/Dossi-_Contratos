# Dossiê Imobiliário — correção para Render

## Como atualizar

1. Descompacte este pacote.
2. Substitua os arquivos do repositório GitHub pelos arquivos deste pacote.
3. Faça o commit.
4. No Render, abra o serviço.
5. Clique em Manual Deploy.
6. Clique em Clear build cache & deploy.
7. Aguarde o status Live.

## Variáveis no Render

OPENAI_API_KEY = sua chave real, sem aspas
OPENAI_MODEL = gpt-5-mini

## Teste

Use primeiro um PDF em que seja possível selecionar e copiar o texto.
Na tabela "Diagnóstico dos arquivos", o documento deve apresentar mais de 100 caracteres.

Quando aparecer "Provável PDF escaneado", a aplicação não consegue ler o conteúdo sem OCR.
